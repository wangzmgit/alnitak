#### 升级需要添加以下接口并配置权限
| 路径                      | 请求方法 | 分组 | 简介             |
| ------------------------- | -------- | ---- | ---------------- |
| /api/v1/upload/checkVideo | POST     | 上传 | 获取视频上传进度 |
| /api/v1/upload/chunkVideo | POST     | 上传 | 上传视频文件分片 |
| /api/v1/upload/mergeVideo | POST     | 上传 | 合并视频文件分片 |

#### 升级需要修改配置文件
1. `max-age`改为`max_age`
2. `max-backups`改为`max_backups`
3. `max-size`改为`max_size`
