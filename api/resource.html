视频资源接口 ​

修改资源标题 ​

请求URL ​

  • /api/v1/resource/modifyTitle

请求方式 ​

  • PUT

请求头 ​

  • Authorization': token
  • "content-type": "application/json",

参数 ​

参数名必选类型说明
id是int资源ID
title是string资源标题

返回示例 ​

json
{
  "code": 200,
  "data": null,
  "msg":"ok"
}

备注 ​

无

删除资源 ​

请求URL ​

  • /api/v1/resource/deleteResource/资源ID

请求方式 ​

  • DELETE

请求头 ​

  • Authorization': token

返回示例 ​

json
{
  "code": 200,
  "data": null,
  "msg":"ok"
}

备注 ​

无
